## 📖 About This Repo

This repository documents my hands-on learning of **C programming** and **low-level security concepts** as I work through *Hacking: The Art of Exploitation* by Jon Erickson.

Each chapter folder contains:
- 📝 `notes.md` — Key concepts, syntax, and takeaways
- 💻 `.c` files — Programs I wrote while studying that chapter

The goal is to deeply understand how computers work at a low level — memory, pointers, processes, and ultimately how vulnerabilities arise and how to think about them like a security professional.
